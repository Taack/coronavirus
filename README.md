# coronavirus
Coronavirus Simulation

# Hypothesis

* Number of infected >> Number of dead people ;
* 2,5 Days to double infected population
* 10 days to suffer from infection (+/- 3 Days)
* Lethality of 0.002

# Usage

`source("~/github/coronavirus/rstudio/italy.r", chdir = T)`
